# Política de Privacidade

**Última Atualização:** 18 de Janeiro de 2026

## 1. Introdução

CleanCalc ("nós", "nosso" ou "o Aplicativo") compromete-se a proteger sua privacidade. Esta Política de Privacidade explica nossas práticas de dados para o aplicativo móvel e web CleanCalc.

## 2. Coleta de Dados

**Não coletamos, transmitimos ou armazenamos quaisquer dados pessoais em servidores externos.**

CleanCalc é um aplicativo apenas frontend. Todo o processamento e armazenamento de dados ocorre localmente no seu dispositivo.

## 3. Informações Armazenadas Localmente

As seguintes informações são armazenadas localmente no seu dispositivo:

- **Tipos de Serviço**: Ofertas de serviço de limpeza personalizadas e preços base
- **Regras de Preços**: Seus fatores de preços e taxas configuradas
- **Orçamentos**: Orçamentos gerados com detalhes da propriedade e cálculos
- **Configurações**: Preferências do aplicativo (tema, idioma, moeda, unidades)
- **Modelos**: Modelos de orçamento salvos para reutilização

## 4. Uso de Dados Locais

Os dados locais são usados exclusivamente para:

- Realizar cálculos de preços
- Salvar suas preferências e configurações
- Manter o histórico de orçamentos
- Permitir a reutilização de modelos

## 5. Compartilhamento de Dados

**Não compartilhamos seus dados com terceiros** porque:

- Todos os dados permanecem no seu dispositivo
- Nenhum dado é transmitido para servidores externos
- Nenhum serviço de análise ou rastreamento é usado
- Nenhuma rede de publicidade tem acesso aos seus dados

## 6. Exportação e Compartilhamento de PDF

Ao exportar ou compartilhar um orçamento como PDF:

- O PDF é gerado localmente no seu dispositivo
- Você controla onde e como o PDF é compartilhado
- Não recebemos cópias de PDFs exportados
- O compartilhamento ocorre através da funcionalidade nativa de compartilhamento do seu dispositivo

## 7. Segurança de Dados

A segurança dos seus dados depende de:

- Configurações de segurança do seu dispositivo
- Proteções do sistema operacional do seu dispositivo
- Controles de acesso físico ao seu dispositivo

Recomendamos:

- Usar criptografia do dispositivo
- Definir uma senha forte para o dispositivo
- Manter seu sistema operacional atualizado

## 8. Retenção de Dados

Os dados permanecem no seu dispositivo até que você:

- Os exclua manualmente através do Aplicativo
- Limpe os dados do Aplicativo através das configurações do dispositivo
- Desinstale o Aplicativo

## 9. Privacidade de Crianças

CleanCalc destina-se ao uso comercial por profissionais de limpeza. Não coletamos intencionalmente dados de crianças menores de 13 anos.

## 10. Seus Direitos

Você tem controle total sobre seus dados:

- **Acesso**: Visualizar todos os dados dentro do Aplicativo
- **Modificar**: Editar ou atualizar qualquer informação armazenada
- **Excluir**: Remover orçamentos individuais ou limpar todos os dados
- **Exportar**: Gerar PDFs de seus orçamentos

## 11. Serviços de Terceiros

CleanCalc não se integra com serviços de terceiros que coletam dados. Toda a funcionalidade é independente.

## 12. Alterações na Política de Privacidade

Podemos atualizar esta Política de Privacidade periodicamente. O uso continuado do Aplicativo após as alterações constitui aceitação da política atualizada.

## 13. Informações Específicas da Plataforma

### Versão Web
- Dados armazenados no armazenamento local do navegador
- Limpar os dados do navegador excluirá os dados do Aplicativo
- Não são usados cookies para rastreamento

### Versão Móvel (iOS/Android)
- Dados armazenados no armazenamento específico do aplicativo
- Acessível apenas pelo Aplicativo
- Removido quando o Aplicativo é desinstalado

### Versão Desktop (macOS/Windows)
- Dados armazenados no diretório de documentos do aplicativo
- Protegido pelas permissões do sistema operacional
